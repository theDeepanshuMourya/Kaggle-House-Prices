# Kaggle-House-Prices
This is a project for Kaggle House Price Prediction Competition - [House Prices: Advanced Regression Techniques](https://www.kaggle.com/c/house-prices-advanced-regression-techniques). The objective of the project is to perform data visulalization techniques to understand the insight of the data. This project aims at applying various Python tools to get a visual understanding of the data and clean it. Also the project uses Python 3.6.
